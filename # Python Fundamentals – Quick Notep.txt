## 8. Control Statements

### if / elif / else

```python
if age >= 18:
    print("Adult")
else:
    print("Minor")
```

### Loops

#### for loop

```python
for i in range(1,6):
    print(i)
```

#### while loop

```python
i = 1
while i <= 5:
    print(i)
    i += 1
```
